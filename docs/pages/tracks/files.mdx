### Files 

Lightrail can access files that are stored locally in your computer for use in arbitrary queries.

**Tokens**: 
1. `/files.path`: Reference a file path - can be used to add the contents of a directory to the Lightrail Knowledge Base. 
![Add contents of a directory to KB](../../assets/demo-add-kb.png)
2. `/files.content`: Read in the contents of a text file
![File Content Demo](../../assets/file-content-demo.png)
